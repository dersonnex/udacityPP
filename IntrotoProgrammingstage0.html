Basic of the web and HTML

How The Web Works

Four pieces of the web are your computer, browser, HTTP and the servers.

Every File type can be found on the Internet .

File Type

The main type is HTML. HTML stands for Hypertext Markup Language.

HTML Tags

HTML elements are what we use to tell a browser how to display content in a web page. They are like boxes. Most HTML elements consist of an opening and a closing tag with some content between them.
Tag is like a container. "b" stands for bold. " i" stands for italics.
